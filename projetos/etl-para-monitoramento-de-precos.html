ETL para monitoramento de preços

Nesse projeto foi feito uma ETL para monitoramento de preços utilizando python e a biblioteca scrapy Para o web scraping foi utilizada a biblioteca Scrapy para extrair dados do Mercado Livre, especificamente tênis de corrida masculino. Foi utilizada a url para realizar a raspagem. Em seguida foi feita a transformação dos dados e inserida no banco SQLite. E por ultimo foi feita um dashboard usando o stremlit.

As ferramentas utilizadas foram:

  • Git, Github;
  • Python, Pandas;
  • scrapy
