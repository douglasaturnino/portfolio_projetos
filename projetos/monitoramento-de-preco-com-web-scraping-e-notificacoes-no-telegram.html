Monitoramento de Preço com Web Scraping e Notificações no Telegram

Este projeto realiza o monitoramento de preços de produtos do mercado livre. Utilizando técnicas de web scraping, o projeto coleta preços e envia notificações no Telegram quando o valor atinge um preço menor que o atual.

As ferramentas utilizadas foram:

  • Python, Pandas;
  • Github, Docker-Compose, uv;
  • Telegram;
  • Web Scraping;
  • Cloud Render.
