Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógico e logica de negócio, junto com ferramentas de desenvolvimento web como o Streamlit e Github, foram usadas para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.O resultado final do projeto foi um painel hospedado em um ambiente Cloud e distribuido através de um link web. O painel pode ser acessado por qualquer dispositivo com internet.

As ferramentas utilizadas foram:

  • Git, Github;
  • Python, Pandas, Matplotlib e Seaborn;
  • Streamli, Streamlit Cloud.
