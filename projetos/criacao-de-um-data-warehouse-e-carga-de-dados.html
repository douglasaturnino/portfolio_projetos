Criação de um Data Warehouse e Carga de Dados

Nosso objetivo era pegar dados do municipais do Rio de Janeiro, utilizando fonte do IBGE e registros de delegacias de polícia no formato csv

Criar bancos de dados dedicados, incluindo o Operational Data Store (ODS) para dados operacionais e o Data Warehouse (DW) para análise dimensional, garantimos que cada aspecto dos dados seja explorado de maneira eficiente e precisa.

As ferramentas utilizadas foram:

  • Git, Github;
  • Python
  • SQLite
